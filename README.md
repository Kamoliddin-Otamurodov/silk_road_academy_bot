# Silk Road Academy Telegram Bot

## Overview
A Telegram bot developed for an educational center (Silk Road Academy).  
This project was built to practice Python backend skills and automate routine communication with students.

## Key Features
- Command-based interaction (menu / basic commands)
- User data storage (example structure provided)
- Webhook-ready deployment (for hosting on a server)

## Tech Stack
- Python
- python-telegram-bot
- Environment variables via `.env` (see `.env.example`)

## Project Structure
- `run.py` — main entry point
- `webhook.py` — webhook setup (for server deployment)
- `.env.example` — sample environment configuration

## Setup (Local)
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies:
   ```bash
